# waypoint

## Privacy Policy                                                                                                                                                  
                                                                                                                                                                   
  *Last updated: 26 March 2026*                                                                                                                                      
                                                                                                                                                                     
  ### Overview                                                                                                                                                       
                                                                                                                                                  
  Waypoint is a Chrome extension that provides a management dashboard overlay for SAP SuccessFactors. This policy explains how the extension handles data.           
   
  ### Data accessed                                                                                                                                                  
                                                                                                                                                                   
  Waypoint reads the following data from SAP SuccessFactors via your organisation's existing WorkZone session:
  - Your profile information and that of your direct reports (name, job title, department, location)
  - Goals and activities
  - Feedback records
  - Performance ratings
  - Promotion case information                                                                                                                                       
   
  ### Data storage                                                                                                                                                   
                                                                                                                                                                   
  - All data remains within your browser
  - A short-term cache (maximum 5 minutes) is stored in your browser's local storage to reduce API calls
  - UI preferences (dark mode, last active tab) are stored in local storage
  - No data is written to any database or external server                                                                                                            
   
  ### Data sharing                                                                                                                                                   
                                                                                                                                                                   
  Waypoint does not collect, transmit, share, or sell any personal data. No analytics, tracking, or telemetry of any kind is included.

  ### Third parties

  Waypoint makes no requests to any third-party servers. All API calls go exclusively to your organisation's SAP SuccessFactors instance via the WorkZone proxy.
